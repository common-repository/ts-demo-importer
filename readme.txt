TS Demo Importer will add customizer and regsiter widgets.

== Description ==

TS Demo Importer will allow you to add features of customizer and widgets. Which will be visible on front page of site.

== Installation ==

1. Upload the `ts_demo_importer.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
